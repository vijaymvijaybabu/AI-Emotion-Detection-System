# 😊 AI Emotion Detection System

An AI-powered **Facial Emotion Detection System** that uses **DeepFace, OpenCV, Flask, and Computer Vision** to recognize human emotions in real time from live webcam streams and uploaded images. The application features an interactive web dashboard with live emotion visualization, confidence scores, emotion history, analytics, and facial expression recognition.

---

## 🚀 Project Overview

Understanding human emotions is an important aspect of Human-Computer Interaction (HCI). This project leverages Artificial Intelligence and Deep Learning to detect facial expressions and classify emotions such as **Happy, Sad, Angry, Fear, Surprise, Disgust, and Neutral**.

The system supports both **live webcam emotion detection** and **image-based emotion analysis**, making it suitable for educational, healthcare, customer experience, and AI research applications.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AI-Emotion-Detection.git

cd AI-Emotion-Detection
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```
